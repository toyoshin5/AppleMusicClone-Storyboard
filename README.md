## 実行方法

Xcodeを起動して command + R

## 良かったところ
 - 静的な箇所の多くをノーコードで作れるところ
 - UIの位置を微調整をしやすい
 - 情報が多いところ

## よくなかったところ
 - 曲名表示などの動的なUIの実装はしんどい
 - UIの再利用にも手間がかかる
 - UIの固定化は結構頭使う
 - 角丸のViewに影をつけるのが難しい
 - コンフリクトしたら詰み
